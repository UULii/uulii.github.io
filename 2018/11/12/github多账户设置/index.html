github多账户设置

发表于 2018-11-12

原文参考链接

清除全局配置

查看

git config --global user.name
git config --global user.email

清除

git config --global --unset user.name
git config --global --unset user.email

密钥

进入目录

cd ~/.ssh

生成密钥

ssh-keygen -t rsa -C "登录邮箱" 对应生成不同的密钥

添加密钥到本地

ssh-add ~/.ssh/密钥私钥名称

验证是否添加到本地

ssh-add -l

生成本地密钥配置文件

touch config

本地密钥配置文件内容

Host github1 // 网站的别名，随意取
HostName github.com // 托管网站的域名
User github1 // 托管网站上的用户名
IdentityFile ~/.ssh/id_rsa_github1 // 使用的密钥文件

Host gitlab2
HostName gitlab.com
User gitlab2
IdentityFile ~/.ssh/id_rsa_gitlab2

公钥添加到托管网站

测试配置是否成功

ssh -T git@网站别名

Coding.net

Host coding
HostName git-ssh.coding.net // 这个域名使用coding官网获取的，不能写coding.net
User liugui
IdentityFile ~/.ssh/id_rsa_coding
Port 443 // 20端口可能被禁用，需要使用443端口
# github,多账户
